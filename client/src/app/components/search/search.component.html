Manually select query to execute:

To INSERT:

Provide table attributes and values, each separated by a space.

For example, if you are inserting into Users table with NAME = Nick, AGE = 30, then type "NAME AGE" in Attributes field and "Nick 30" in Values field.

To DELETE:

Provide table attributes and primary key.

For example, if you are deleting from Users table with SSN = 123456789, then type "SSN" in Attributes field and "123456789" in Values field.

To UPDATE:

Provide table attributes and values.

For example, if you are updating Users table for user with SSN = 123456789, and you want to update the AGE to 45, then type "SSN AGE" in Attributes field and "123456789 35" in Values field.
